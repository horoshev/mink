# mink
Hyperlink minifier

## General
Реализация состоит из backend(dotnet 5.0) и frontend(react). В качестве базы данных используется SQLite.

## Запуск
backend (Mink.Api) - `dotnet run` \
frontend (Mink.WebUI) - `npm start`
